# Android Helper Placeholder

This directory is reserved for a future Hermes Mock Location Helper App.

Phase 1 does not include Android app implementation. The planned helper will receive:

```text
com.hermes.mocklocation.SET
```

with `lat`, `lon`, and `accuracy` extras, then apply mock location only in authorized test environments.

Planned package name:

```text
com.hermes.mocklocation
```
